# 🚗 نظام إدارة مصروفات السيارات

## 📋 **نظرة عامة**

نظام شامل ومتطور لإدارة مصروفات السيارات مع تكامل سحابي باستخدام Supabase. يوفر النظام واجهة سهلة الاستخدام لتتبع جميع المصروفات المتعلقة بالسيارات مع إمكانيات التحليل والتقارير المتقدمة.

## 🌟 **الميزات الرئيسية**

### 🔐 **نظام المصادقة المتقدم**
- تسجيل الدخول بالبريد الإلكتروني
- إنشاء حساب جديد
- تسجيل الدخول بـ Google OAuth
- العمل بدون حساب (وضع محلي)

### 📊 **إدارة شاملة للبيانات**
- إدارة معلومات السيارات
- تتبع بيانات الموظفين
- تسجيل جميع أنواع المصروفات
- رفع وإدارة المرفقات

### ☁️ **التخزين السحابي الآمن**
- تكامل مع Supabase
- مزامنة تلقائية للبيانات
- نسخ احتياطية آمنة
- Row Level Security (RLS)

### 📱 **تطبيق ويب تقدمي (PWA)**
- يعمل بدون إنترنت
- قابل للتثبيت على الأجهزة
- واجهة متجاوبة مع جميع الشاشات
- أداء سريع ومحسن

## 🚀 **البدء السريع**

### **1. الوصول للنظام**
```
https://mohamedmarouf55.github.io/car-expense-manager/
```

### **2. إعداد Supabase (للتخزين السحابي)**
راجع ملف [`SUPABASE_SETUP.md`](./SUPABASE_SETUP.md) للحصول على دليل مفصل.

### **3. استخدام النظام**
- **مع حساب:** تسجيل دخول للحصول على التخزين السحابي
- **بدون حساب:** العمل محلياً فقط

## 🛠️ **التقنيات المستخدمة**

### **Frontend**
- HTML5, CSS3, JavaScript (ES6+)
- Tailwind CSS للتصميم
- Font Awesome للأيقونات
- Chart.js للرسوم البيانية

### **Backend & Database**
- Supabase (PostgreSQL)
- Row Level Security
- Real-time subscriptions
- File storage

### **PWA Features**
- Service Worker
- Web App Manifest
- Offline functionality

## 📁 **هيكل المشروع**

```
car-expense-manager/
├── index.html                 # الواجهة الرئيسية
├── app.js                     # منطق التطبيق الأساسي
├── supabase-config.js         # إعدادات Supabase
├── supabase-integration.js    # تكامل Supabase
├── database-schema-simple.sql # هيكل قاعدة البيانات
├── manifest.json              # PWA manifest
├── sw.js                      # Service Worker
├── browserconfig.xml          # إعدادات المتصفح
├── README.md                  # هذا الملف
└── SUPABASE_SETUP.md         # دليل إعداد Supabase
```

## ⚙️ **الإعداد والتطوير**

### **متطلبات النظام**
- متصفح حديث يدعم ES6+
- اتصال بالإنترنت (للتخزين السحابي)
- حساب Supabase (اختياري)

### **التثبيت المحلي**
```bash
# استنساخ المستودع
git clone https://github.com/mohamedmarouf55/car-expense-manager.git

# الانتقال للمجلد
cd car-expense-manager

# فتح في متصفح محلي
# يمكن استخدام Live Server في VS Code
```

### **إعداد Supabase**
1. إنشاء مشروع جديد في [Supabase](https://supabase.com)
2. تنفيذ ملف `database-schema-simple.sql`
3. تحديث مفاتيح API في `supabase-config.js`
4. تفعيل المصادقة المطلوبة
